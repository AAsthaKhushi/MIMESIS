# MIMESIS - Modern Political Art Gallery

A modern web application showcasing political art and providing resources for artists and art enthusiasts. Built with React, TypeScript, and Vite.

## Features

- Interactive art gallery
- Artist profiles and collections
- Educational resources about political art
- Timeline of political art movements
- Responsive design with modern UI

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- Lucide React Icons

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/AAsthaKhushi/MIMESIS.git
cd MIMESIS
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
# or
yarn build
```
